# בדיקה גרמטית - כסף שמגיע לך

## עקרון מרכזי: פנייה ביחיד (אתה/את)

לאחר סריקה מקיפה של כל האתר, החלטנו על **פנייה ביחיד** בכל האתר:
- ✅ "אתה זכאי" (לא "אתם זכאים")
- ✅ "יש לך" (לא "יש לכם")
- ✅ "בואו נתחיל" → "בוא נתחיל"

זה יוצר חיבור אישי יותר עם המשתמש.

---

## תיקונים לפי עמוד

### Home.tsx
**בעיות שנמצאו:**
1. ❌ "יש הרבה עזרה שממדינה נותנת" → ✅ "המדינה נותנת הרבה עזרה"
2. ❌ "באנו נבדוק ביחד מה מגיע לכם" → ✅ "בוא נבדוק ביחד מה מגיע לך"
3. ❌ "זה לוקח רק 2 דקות" → ✅ "זה לוקח רק שתי דקות"
4. ❌ "אנחנו נראה לכם בדיוק מה לעשות" → ✅ "אנחנו נראה לך בדיוק מה לעשות"

**טון:**
- הטקסט הנוכחי טוב, אבל צריך להיות יותר חם ופחות פורמלי

---

### Questionnaire.tsx
**בעיות שנמצאו:**
1. ❌ "בואו נתחיל!" → ✅ "בוא נתחיל!"
2. ❌ "שמרנו בשבילכם" → ✅ "שמרנו בשבילך"
3. Progress bar: "שלב X מתוך 4" - ✅ טוב

**טון:**
- השאלות ברורות, אבל צריך להוסיף עידוד

---

### Results.tsx
**בעיות שנמצאו:**
1. ❌ "מצאנו לכם X תוכניות" → ✅ "מצאנו לך X תוכניות"
2. ❌ "אתם זכאים" → ✅ "אתה זכאי / את זכאית"
3. ❌ "לפרטים נוספים" → ✅ "למידע נוסף" (יותר פשוט)

**טון:**
- צריך להוסיף עידוד: "כל הכבוד! מצאנו לך..."

---

### UrgentHelp.tsx
**בעיות שנמצאו:**
1. ❌ "אנחנו כאן בשבילכם" → ✅ "אנחנו כאן בשבילך"
2. ❌ "אל תהססו" → ✅ "אל תהסס"
3. ❌ "צרו קשר" → ✅ "צור קשר"

**טון:**
- טוב, אבל צריך להיות יותר מעודד ופחות פורמלי

---

### PersonalStory.tsx
**בעיות שנמצאו:**
1. ❌ "למה בניתי את זה?" → ✅ "למה בניתי את המערכת הזו?"
2. Mixing של יחיד/רבים בסיפור

**טון:**
- הסיפור טוב, אבל צריך להיות יותר אישי

## סיכום תיקונים נדרשים

### 1. עקביות יחיד/רבים
- **החלטה:** פנייה ביחיד בכל האתר
- **דוגמאות:**
  - "לכם" → "לך"
  - "אתם" → "אתה/את"
  - "בואו" → "בוא"
  - "צרו" → "צור"

### 2. תחביר
- משפטים קצרים וברורים
- הימנעות ממבנים מסורבלים
- שימוש בשפה פשוטה

### 3. טון
- **חם ואנושי** - לא רובוטי
- **מעודד** - "כל הכבוד!", "מצוין!"
- **אופטימי** - "יש הרבה עזרה זמינה"
- **אישי** - "אני כאן לעזור לך"

### 4. מספרים
- מספרים 1-10 במילים: "שתי דקות" (לא "2 דקות")
- מספרים 11+ בספרות: "13 תוכניות"

## תיקונים ספציפיים

### Home.tsx - תיקונים
```tsx
// ❌ לפני
"יש הרבה עזרה שממדינה נותנת."
"באנו נבדוק ביחד מה מגיע לכם."

// ✅ אחרי
"המדינה נותנת הרבה עזרה שאולי לא ידעת עליה."
"בוא נבדוק ביחד מה מגיע לך."
```

### Questionnaire.tsx - תיקונים
```tsx
// ❌ לפני
"בואו נתחיל!"
"שמרנו בשבילכם"

// ✅ אחרי
"בוא נתחיל!"
"שמרנו בשבילך את התשובה"
```

### Results.tsx - תיקונים
```tsx
// ❌ לפני
"מצאנו לכם 13 תוכניות סיוע"
"אתם זכאים"

// ✅ אחרי
"כל הכבוד! מצאנו לך 13 תוכניות סיוע"
"אתה זכאי / את זכאית"
```

### UrgentHelp.tsx - תיקונים
```tsx
// ❌ לפני
"אנחנו כאן בשבילכם 24/7"
"אל תהססו לפנות"

// ✅ אחרי
"אנחנו כאן בשבילך 24 שעות ביממה"
"אל תהסס לפנות - זה בסדר לבקש עזרה"
```
